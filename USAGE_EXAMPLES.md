# 📖 דוגמאות שימוש מפורטות

## 🚀 הפעלה מהירה

### התקנה בסיסית
```bash
# שכפול הפרויקט
git clone <repository_url>
cd telegram_todo_bot

# התקנת ספריות
pip install -r requirements.txt

# הגדרת טוקן
export TELEGRAM_BOT_TOKEN="your_bot_token_here"

# הרצה בסיסית
python main.py
```

### הרצה מתקדמת
```bash
# הרצה עם כל התכונות המתקדמות
python run_bot.py

# הרצה במצב ייצור
python run_bot.py --production

# הרצה ללא תכונות מתקדמות
python run_bot.py --no-enhanced

# יצירת שירות systemd
python run_bot.py --create-service
```

## 🔧 תכונות לפיתוח מתקדם

### שימוש ב-API פנימי
```python
# דוגמא לשימוש בבוט כמודול
from main import TodoBot

bot = TodoBot("your_token")
bot.init_database()

# הוספת משימה פרוגרמטית
task_id = bot.add_task(
    user_id=123456789,
    content="משימה אוטומטית מהמערכת",
    category="עבודה"
)

# קבלת סטטיסטיקות
stats = bot.get_task_summary(123456789)
print(f"סך משימות פתוחות: {sum(stats.values())}")
```

### יצירת תגיות אוטומטיות
```python
# הוספת משימה עם תגיות
content = "לסיים פרויקט האתר #דחוף #לקוח #פיתוח"
# הבוט יזהה אוטומטית את התגיות: דחוף, לקוח, פיתוח

# חיפוש לפי תגיות
results = bot.search_tasks(user_id, "#דחוף")
```

### אינטגרציה עם מערכות חיצוניות
```python
# דוגמא לשליחת משימה מגוגל קלנדר
import json
from datetime import datetime

def create_task_from_calendar_event(event_data):
    task_content = f"{event_data['summary']} - {event_data['start']['dateTime']}"
    
    # זיהוי קטגוריה אוטומטי לפי מילות מפתח
    if any(word in task_content.lower() for word in ['פגישה', 'meeting']):
        category = 'עבודה'
    elif any(word in task_content.lower() for word in ['רופא', 'קניות']):
        category = 'אישי'
    else:
        category = 'כללי'
    
    return bot.add_task(user_id, task_content, category)
```
